Evolution of The Web

1969

Charles Goldfarb, Edward Mosher and Raymond Lorie developed "GML"

[ whose surname initials were used to make up the term GML ].

1980

Guido van Rossum conceived Python.

1985

October 4:

The Free Software Foundation is founded by

Richard Matthew Stallman.

1986

October:

SGML (The Standard Generalized Markup Language)

was accepted followed by a minor Technical Corrigendum.

1987

Larry Wall developed "Perl".

there are various backronyms in use, the most well-known being

"Practical Extraction and Reporting Language".

1989

Tim Berners-Lee wrote a memo

proposing an Internet-based hypertext system.

December:

Python implementation start.

1990

Tim Berners-Lee specified HTML (HyperText Markup Language)

and wrote the server software

[ Berners-Lee considered HTML to be an application of SGML ].

Ruby its developed by Yukihiro "Matz" Matsumoto.

two names were proposed: "Coral" and "Ruby",

Matsumoto later noted that "Ruby"

was the birthstone of one of his colleagues.

December 25:

Berners-Lee made the first web browser "WorldWideWeb".

[ later renamed to Nexus to avoid confusion ].

1992

April 15:

Born "Erwise" the first web browser with a graphical user interface.

[ project initiated by Robert Cailliau ]

1993

January 23:

Born "Mosaic" the web browser credited by popularizing

the World Wide Web.

June 8:

Born The first Microsoft Windows browser under the name of "Cello".

1994

Rasmus Lerdorf created "PHP".

originally stood for Personal Home Page, it now stands for

PHP: Hypertext Preprocessor

[ which is a recursive backronym ].

October:

It's founded by Tim Berners-Lee the World Wide Web Consortium (W3C),

the main international standards organization

for the World Wide Web.

October 10:

CSS (Cascading Style Sheets)

was proposed by HÅkon Wium Lie.

December 15:

Netscape Navigator (based on Mosaic Web Browser)

made his firsts steps.

1995

The Begining of "Opera" (Web Browser).

HTML 2.0 Happens!

The first HTML specification intended to be treated as

a standard against which future implementations should be based.

It's developed "Mocha" by Brendan Eich while working on Netscape.

although it was officially called LiveScript.

later on september it's renamed to Javascript

[ even though it wasn't related to Java ].

March 1:

Born "Yahoo! Search".

The word "Yahoo" is an acronym for "Yet Another Hierarchical Officious Oracle".

August 16:

Internet Explorer was released.

December 15:

It's Founded AltaVista (the web search engine).

[ whom later in July 8 of 2003 was purchased by Yahoo! ].

1996

CSS 1 was released

Bert Bos' proposal was influential also he became co-author of CSS 1

and is regarded as co-creator of CSS.

January:

As a research project by Larry Page and Sergey Brin born "Google".

August-November:

The first Working Draft of an XML specification was published.

XML (Extensible Markup Language) an application profile of SGML.

October 14:

KDE created "Konqueror" under the KHTML layout engine.

1997

December:

HTML 4.0 was published as a W3C Recommendation

with three variations:

-- Strict, Transitional and Frameset --

1998

PGML (Precision Graphics Markup Language)

it's submitted to the W3C.

[ that was not adopted as a recommendation].

February 10:

XML 1.0 became a W3C Recommendation.

April:

MathML 1 (Mathematical Markup Language)

was released as a W3C recommendation as the first

XML language to be recommended by the W3C.

1999

HTML 4.01 was published!

After PGML wasnt adopted as a recomendation

start the development of SVG (Scalable Vector Graphic).

June:

The earliest CSS 3 drafts were published.

2000

January 26:

XHTML 1.0 became a W3C Recommendation.

July:

C# was publicly announced!! they use the number sign because the sharp symbol

is not available on the standard keyboard. "sharp" resembles a ligature of

four "+" symbols implying that the language is an increment of C++.

2001

SVG 1.0 became a W3C Recomendation.

January 15:

Jimmy Wales and Larry Sanger launched Wikipedia.

February:

Appeared MathML 2.0.

May 31:

XHTML 1.1 became a W3C Recommendation.

2002

September 23:

Under the name of Phoneix born Firefox.

2003

January 7:

Safari is released as a public beta.

It was based on Apple's internal fork of the KHTML rendering engine

[ WebKit ].

October:

MathML 2.0 was published as the final release

by the W3C math working group.

May 27:

Born Wordpress by Matt Mullenweg and Mike Little.

2004

John Gruber with Aaron Swartz created the "Markdown language".

Development of HTML5 began.

2005

June:

WebKit was released as open source.

2006

Hampton Catlin created "Haml" (HTML Abstraction Markup Language).

August 26:

Influenced by cssQuery Was Released Jquery 1.0

by John Resig at BarCamp.

2007

Hampton Catlin created "Sass" (Syntactically Awesome Stylesheets).

2008

October 28:

HTML5 is completed and standardized.

September 2:

Google Chrome was released as a beta version.

[ Under the WebKit layout engine that was already open source ]

2009

Born "Less"!

a CSS builder system by the hand of Alexis Sellier and influenced by "Sass".

A ninth draft of XHTML 2.0 was expected to appear,

but on July 2, the W3C decided to let

the XHTML2 Working Group charter expire by that year's end.

Born AngularJS by Miško Hevery and Adam Abrons

at Brat Tech LLC.

2010

October:

MathML 3 specification was released as a

W3C Recommendation.

2011

June 7:

CSS 2.1 was published as a W3C Recommendation.

2013

February 12:

Opera announced it would drop

its own Presto engine in favour of WebKit.

April 3:

Google announced that it would form

a new rendering engine known as Blink.

the same day, Opera confirmed that it would follow Google.

April 18:

Jquery Dropped support for IE[6-8] for performance

improvements and reduction in filesize,

releasing the version 2.03.

2014

October:

HTML 5 was published as a W3C Recommendation.

NOW

W3C is still working on the standard of XHTML5.

But the story has just begun...
